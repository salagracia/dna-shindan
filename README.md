# DNA診断 v3.1 — サラ・グラシアアカデミー監修

サラさんのLINE登録プレゼント用「**人生再起動のためのDNA設計図**」生成アプリ。

東洋・西洋の占術 + 現代心理学 = 8ページの個人レポート

---

## 機能

### 入力（3分で完了）
- 氏名（漢字＋カタカナ）
- 生年月日・出生時間・出生地
- 性格診断 8問（MBTI）
- ウェルスダイナミクス 1問

### 占術データ（9種）
1. **数秘術**：ライフパス・誕生数・運命数
2. **西洋占星術**：太陽星座・月星座・アセンダント（NASA天体計算）
3. **九星気学**：本命星・月命星
4. **四柱推命**：年柱・月柱・日柱・時柱（万年暦ベース）
5. **動物キャラ**（個性心理学60種＋12種）
6. **算命学**：主星（日干×月支120パターン早見表）
7. **帝王学**：8種の運命型
8. **16パーソナリティ**：詳細プロファイル
9. **ウェルスダイナミクス**：詳細プロファイル

### 出力（8ページPDF）
- p1：表紙
- p2：占術データ一覧（17項目）
- p3：16パーソナリティ詳細（強み・弱点・人間関係・キャリア・チャレンジ）
- p4：ウェルスダイナミクス詳細（運気戦略含む）
- p5：総合診断（才能・使命・落とし穴）
- p6：相性診断（恋愛・ビジネスパートナー）
- p7：運勢サイクル（**天中殺の具体年・3年間のキーワード**）
- p8：これからの行動指針（取るべき3つ・避けるべき3つ）

## 使い方

### ローカル実行

```bash
cd D:/CCBooster/dna-shindan-v2
pip install -r requirements.txt
python -m streamlit run app.py
```

→ http://localhost:8501

### スクリプトで直接生成

```bash
python main.py
```

→ `output/DNA診断_山岡サラ_v3.1_YYYYMMDD.pdf` が生成される
